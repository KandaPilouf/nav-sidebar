# Simple responsive navbar

Simple navbar using Javascript to hide/show and responsive to mobile and tablet format

## Learnings
- Using Javascript and changing DOM element
- Making functions to hide and show (changing display of the navbar elements)
- Media queries for different screen sizes (When to show the burger icon and different sizes for the navbar)

## Tech
- HTML5  
- CSS
- JavaScript
